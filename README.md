# all-mobile-notification

## Settings

### Android

Set the following properties in `notifier.properties` or `System.properties`:
 - `android.title`
 - `android.key`

### iOS

Set the following properties in `notifier.properties` or `System.properties`:
 - `ios.cert.path`
 - `ios.cert.pass`

### Windows

Set the following properties in `notifier.properties` or `System.properties`:
 - `windows.sid`
 - `windows.secret`
