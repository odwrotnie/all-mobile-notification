# all-mobile-notification

## Settings

### iOS

Place the Apple certificate in `ios-cert.p12` in the classpath.

### Android

Set the following properties in `notifier.properties' or `System.properties`:
 - `android.title`
 - `android.key`

### Windows

Set the following properties in `notifier.properties' or `System.properties`:
 - `windows.sid`
 - `windows.secret`
